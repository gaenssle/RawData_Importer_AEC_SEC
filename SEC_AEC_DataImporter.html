SEC and AEC Data Importer

Created for Chromeleon by A.L.O. Gaenssle, 2022

1. Introduction

The SEC and AEC (Peak) Data Importer are small programs I have written in Python 3.5. All three work after the same principle and simply differ in the type of files they import. They import all text files in a folder and accumulate all the samples into a set of export files. Since the programs don't have a user interface they can only be run using the terminal and they only import files from SEC, Dionex (Chromeleon) or any file with the same layout.

2. Install Python

  • Download and install Python from python.org
    • I have version 3.5 but the newest one should work as well
    • Make sure that you tick the box with "add Python to PATH" or it won't recognize it later

3. Setup environment

  • Create the following folder structure for your SEC/AEC files:
    • Main folder (e.g. SEC)
    • Sub-folders (e.g. SEC1_6/8/19)
    • Sub-subfolders (e.g. RawData)
  • Set up an easy access to the program
    • Copy the program (e.g. ImportSEC.py) into your main folder
    • Make a shortcut of the terminal (cmd) in your main folder, right-click on the shortcut, select properties. Go to the tab "Shortcut" and change the path in "Start in:" to your main folder
  • Copy your files to the sub-sub folders
    • Make sure that there are no other .txt files in those folders or it will try to import them as well and crash
    • Make sure the files are in the order you want. This works best if the file names start with a number
    • For SEC, the program adds a leading 0 for samples with a single leading digit

4. Terminal

Since the program don't have a user interface, everything has to be conducted using the terminal (cmd). The terminal is a pure text based window so your mouse is useless in it. However, you can use arrow keys for navigation and the tab key for auto-completion. After typing in the text input hit enter to conduct the command. If you want to kill the program, use CTRL+C.

5. SEC Data Importer

This program imports files from the SEC and combines them into three files, being information, raw data and elution data.

  • Open the terminal and write:

    python ImportSEC.py

  • It will ask you for your folder, thus write then something like:

    SEC1\RawData

  • Then it will list all the .txt files that are going to be imported. Confirm it or deny and type in a new path
  • It will create three output files in the sub-folder:
    • Information: Summary of estimated weights and shift in marker elution volume
    • RawData: The raw, unmodified data in a table
    • ElutionData: The data from the second graph which are limited to the selected range, corrected for both baseline and marker position
  • You can open the files in Excel by dragging the file from the explorer into an Excel window (Columns are separated by tabs)

6. AEC Data Importer

This program imports files from our Dionex and accumulates the raw data into a single file.

  • In Chromeleon, go to Report Designer and create a report (choose default)
  • Export the raw data from our Dionex in Chromeleon
    • Select the samples in the Injection List> right click> Export…>Text format (RawData) (*.txt)
    • Select Parent Folder to set where the files should be saved
    • You might need to move the samples or rename the folder to set it up for the importer
  • Open the terminal and write:

    python ImportAEC.py

  • It will ask you for your folder, thus write then something like:

    AEC1\RawData

  • Then it will list all the .txt files that are going to be imported. Confirm it or deny and type in a new path
  • It will create an output file called ..._RawData.txt which can be opened in e.g. Excel

7. AEC Peak Importer

This program imports the peak data files from our Dionex and accumulated them into three output files, being Area, RelativeArea and Height.

  • Export the raw data from our Dionex in Chromeleon
    • Select the samples in the Injection List> right click> Export…>Text format (Results) (*.txt)
    • Click on the button next to the selection and deselect everything in the new window except Integration
    • Select Parent Folder to set where the files should be saved
    • You might need to move the samples or rename the folder to set it up for the importer
  • Open the terminal and write:

    python ImportAEC_Peaks.py

  • It will ask you for your folder, thus write then something like:

    AEC1\PeakResults

  • Then it will list all the .txt files that are going to be imported. Confirm it or deny and type in a new path
  • You need to enter a minimum area to only export peaks which are larger than that, set to 0 if you want to export all
  • Then, enter the minimum time to exclude any initial peaks by buffer, injection, etc, set to 0 if you want all peak
  • Finally, enter the precision you want the time to be rounded to; this compensates for small shifts and reduces the amount of empty lines but be careful as a too high value will delete some of your peaks (only one per time per sample)
  • It will create three very similar output files in the sub-folder, being Area, Height and RelativeArea for the respective values accumulated in the file
  • You can open the files in Excel by dragging the file from the explorer into an Excel window (Columns are separated by tabs)
